# 🌿 EaseUp

## Sobre o Projeto

O EaseUp é uma plataforma web desenvolvida com o objetivo de promover o bem-estar emocional, o autocuidado e hábitos saudáveis por meio de práticas simples e acessíveis.

A aplicação disponibiliza meditações guiadas, exercícios de respiração, biblioteca musical para relaxamento e acompanhamento do progresso do usuário, oferecendo um ambiente acolhedor para momentos de pausa e cuidado pessoal.

Este projeto foi desenvolvido como parte da Atividade Extensionista III do curso de Engenharia de Software.

## ODS Atendido

### ODS 3 – Saúde e Bem-Estar

O EaseUp está alinhado ao Objetivo de Desenvolvimento Sustentável 3 da Organização das Nações Unidas (ONU), que busca assegurar uma vida saudável e promover o bem-estar para todas as pessoas.

A plataforma contribui para esse objetivo ao incentivar práticas de relaxamento, gerenciamento do estresse e autocuidado.

## Evolução do Projeto

O EaseUp foi iniciado na Atividade Extensionista I e aprimorado ao longo das Atividades Extensionistas II e III.

Principais evoluções implementadas:

- Integração de conteúdos de áudio para meditações;
- Biblioteca musical para relaxamento;
- Sistema de favoritos com persistência local;
- Estatísticas de uso do usuário;
- Melhorias de acessibilidade;
- Personalização do nome do usuário;
- Feedback visual por notificações (toast);
- Feedback sonoro para exercícios de respiração;
- Controle de configuração utilizando Git e GitHub.
